#  Task 3: Deep Dive Analysis & Interactive Dashboard

##  Objective
Perform deep analysis and build an interactive dashboard to explore business insights.

##  Work Done
- Defined key KPIs (Sales, Profit, Customers, Avg Sales)
- Performed segmentation analysis (city, gender, age group)
- Built an interactive dashboard using Tableau
- Added filters for dynamic exploration

##  Dashboard Features
- KPI cards (Total Sales, Profit, Avg Sales, Customers)
- Sales trend over time
- Sales by city
- Customer distribution by gender
- Interactive filters (city, gender)

##  Tools Used
- Tableau
- Python (for preprocessing)

##  Outcome
Created a dynamic dashboard that allows users to explore data and uncover patterns effectively.
